PASOS PARA INSTALAR TODO EN UNA NUEVA COMPUTADORA:
═══════════════════════════════════════════════════════════════════════════════

1. Descarga el proyecto desde GitHub como ZIP
   (Code -> Download ZIP)

2. Extrae el ZIP completo en la computadora del doctor
   Ejemplo: C:\AutoDockVina

3. Abre el archivo:
   C:\AutoDockVina\INSTALAR_TODO.bat
   (o la ruta donde hayas extraido el ZIP)

4. El instalador hará automáticamente:
   ✓ Instalar Python 3.11
   ✓ Crear un entorno virtual
   ✓ Instalar todas las librerías (Flask, BioPython, etc.)
   ✓ Intentar instalar MGLTools (si hay internet)
   ✓ Crear iconos en el Escritorio

5. Espera a que termine (5-15 minutos según la PC)
   NO CIERRES LA VENTANA HASTA QUE DIGA "¡LISTO!"

6. Cuando termine, encontrarás en el Escritorio:
   • Docking Molecular - Web ........... (interfaz principal)
   • AutoDockTools 1.5.6 ............... (preparación de moléculas)
   • PMV 1.5.6 ......................... (visualizador 3D)
   • Vision 1.5.6 ...................... (análisis de moléculas)
   • CADD 1.5.6 ........................ (herramientas científicas)


═══════════════════════════════════════════════════════════════════════════════
¿QUÉ HACE CADA PROGRAMA?
═══════════════════════════════════════════════════════════════════════════════

  🔬 Docking Molecular - Web
     → Interfaz web completa (lo más fácil de usar)
     → Sube moléculas, configura parámetros y ejecuta docking
     → Ver resultados en navegador

  🛠️ AutoDockTools 1.5.6
     → Prepara archivos PDB para docking
     → Convierte a PDBQT
     → Configura caja de docking (grid box)

  👁️ PMV (Python Molecular Viewer)
     → Visualiza moléculas en 3D
     → Ver estructura de proteínas y ligandos

  🔍 Vision & CADD
     → Análisis adicionales de moléculas

═══════════════════════════════════════════════════════════════════════════════
SI ALGO FALLA:
═══════════════════════════════════════════════════════════════════════════════

1. Cierra todas las ventanas
2. Ejecuta nuevamente INSTALAR_TODO.bat
3. Si la descarga de MGLTools falla:
   - Aún puedes usar el sistema web (no lo necesita)
   - Descárgalo manualmente desde: https://ccsb.scripps.edu/mgltools/
   - O usa la línea de comandos para preparar moléculas

4. Si Python no se instala:
   - Descarga manualmente: https://www.python.org/downloads/
   - Marca la opción "Add Python to PATH"
   - Vuelve a ejecutar INSTALAR_TODO.bat

PREGUNTAS FRECUENTES:
═══════════════════════════════════════════════════════════════════════════════

P: ¿Puedo usar esto sin internet?
R: Sí, pero necesitas internet SOLO la primera vez para descargar MGLTools.
   Después, todo funciona sin conexión.

P: ¿Cuánto espacio ocupará todo?
R: ~500 MB en total (Python + librerías + herramientas)

P: ¿Es seguro ejecutar los .bat?
R: Sí, son scripts de instalación automática seguros.

P: ¿Qué pasa si desconecto el USB durante la instalación?
R: La instalación se guarda en la PC, no en el USB. El USB solo es
   el medio de transporte.

P: ¿Puedo copiar el USB a otra PC?
R: Sí, es completamente portable. Cada PC instala independientemente.
